# AGENTS

## Always Do

- When asked to prepare the latest commit for a PR:
  - Rename the latest local `WIP` commit to: `<one-line summary>`.
  - Add a new line with short phrases summarizing the main changes in that PR.
- Use snake_case for all Python names, including keys passed to QWeb templates.
- Follow Odoo 19 best practices for architecture, ORM usage, security, QWeb/XML patterns, and testing.
- Treat [DESIGN.md](DESIGN.md) as the canonical design language document for the website. Use it to guide visual, layout, typography, and component decisions unless the user explicitly overrides it.
